Bugs/Issues
Discover the current bugs and issues in the Equinox Bundles component. Note that a large amount of the activity goes on in Bugzilla. Configure your Bugzilla account email preferences to watch the equinox.bundles-inbox@eclipse.org user to monitor bug activity.

CVS
The Equinox Bundles code repository is part of the normal Eclipse project repository and is available via web access or direct CVS access. For direct access, use anonymous login to the CVS repo

:pserver:anonymous@dev.eclipse.org/eclipse

The Equinox work goes on in various folders as appropriate for the bundles being developed.
