Getting and running Equinox

To get, install and use the Equinox runtime, follow the steps below.

  1. Download and unzip an Eclipse 3.0 build from 20030924 or later
  2. Download the current Equinox runtime and unzip over the previously installed Eclipse build. When prompted to overwrite, answer Yes to all.
    Note that this destructively modifies the original Eclipse install. The only option for reversing this operation is to re-unzip the original Eclipse installation.
  3. Run Eclipse as normal

For more information see the Users Guide.
