Getting and Running Equinox Prototypes

The Equinox prototypes for Phase 1 have been released as part of Eclipse 3.0.

Latest Downloads
Layering Downloads

 
Option 1 - Layering Update Site (preferred option)

  • Install the Eclipse 3.1 I20041201 build
  • Select Help > Software Updates > Find and Install...
  • In the Install/Update dialog, select Search for new features to install
  • On the next page, click the New Remote Site button
  • Enter Equinox Layering and http://download.eclipse.org/technology/equinox/update-site-20041201/ in the New Update Site dialog
  • Expand Equiniox Layering in the tree and check Equinox I20041201 build
  • On the next page, check the Eclispe Platform and Eclipse Plug-in Development Environment features, then complete the Install
  • A dialog will inform you that you must restart Eclipse, click yes to restart
