Dynamic Plug-ins

The goal of the DynamicPlugins work is to make writing, maintaining and using dynamic plug-ins both easier and more failsafe. An overview of the plug-in states is given here.The main issues involved are:

  • Deactivating plugins whose main problem is
    • managing classloaders and loaded classes
  • How to be a good dynamic plugin
  • Mechanisms to support being a good dynamic plug-in
