Mission Statement

The Framework incubator is setup as a staging area for changes to the Equinox OSGi framework project. This work area is primarily targetted to the investigation of major structural changes to the framework. For example, changing the set of bundles that make up the framework and its fragments. Having a distinct sandbox allows this investigation to proceed without cluttering the main repository with speculative projects.

Active investigators

In general all of the Framework component committers will be active in this incubator.
